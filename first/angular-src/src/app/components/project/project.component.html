사용자가 기사와 기자를 검색해 원하는 기사를 빠르게 찾을 수 있게 함
성향에 맞는 기사를 우선적으로 노출
기사와 기자를 평가할 수 있는 시스템 제공
기사에 대한 댓글 기능

기사나 기자도 원하는 취향따라 구독하면 좋지 않을까라는 생각에서 출발
별점이나 평점 등 평가 시스템을 도입하면 좋을 것이라 생각

구독기능을 통해 사용자가 원하는 성향의 정보를 큰 수고 없이 빠르게 찾고 효율적으로 소비할 수 있을 것이라 기대
기사를 평가할 수 있는 시스템을 통해 좋은 기사인지 쉽게 확인할 수 있게 함

프론트엔드: Angular
백엔드: Node.js
DBMS: MongoDB
